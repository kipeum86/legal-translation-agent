# Korean (한국어) Legal Translation Guide

## Register & Formality

| Document Type | Register | Sentence Ending |
|---|---|---|
| 법률/법령 (Statutes) | **한다체** | ~한다, ~하여야 한다 |
| 계약서 (Contracts) | **한다체** | ~한다, ~할 수 있다 |
| 약관/이용약관 (Terms of Service) | **합니다체** or **한다체** | ~합니다, ~됩니다 |
| 개인정보처리방침 (Privacy Policy) | **합니다체** | ~합니다, ~않습니다 |
| 판결문 (Court decisions) | **한다체** | ~한다 |

**Critical rule**: NEVER mix 한다체 and 합니다체 within a single document.

### Standard Legal Sentence Patterns
- **~한다** (declarative): "이 법은 ...을 규정함을 목적으로 한다."
- **~하여야 한다** (mandatory): "사업자는 ...을 고지하여야 한다."
- **~할 수 있다** (permissive): "위원회는 ...을 명할 수 있다."
- **~하여서는 아니 된다** (prohibition): "...하여서는 아니 된다."
- **~으로 본다** (deemed as): "동의한 것으로 본다."
- **~에 따른다** (in accordance with): "관계 법령에 따른다."

## Article Numbering Convention

### Full Hierarchy (largest → smallest)
```
편(編) Part > 장(章) Chapter > 절(節) Section > 관(款) Subsection
> 조(條) Article > 항(項) Paragraph > 호(號) Item > 목(目) Sub-item
```

| Level | Format | Example |
|---|---|---|
| **조** (Article) | 제X조 + title in () | 제1조(목적) |
| **항** (Paragraph) | ①②③④⑤... (circled numbers) | ① 이 법은... |
| **호** (Item) | 1. 2. 3. (Arabic numerals + period) | 1. "회사"란... |
| **목** (Sub-item) | 가. 나. 다. 라. ... (Korean alphabet) | 가. 국내 거주자 |

### Critical Rules
- **항**: Uses circled numbers ①②③. If only ONE paragraph in an article, no number displayed.
- **호**: Arabic numerals with period: 1. 2. 3. — NOT (1) (2) (3)
- **목**: Korean alphabet WITHOUT 제 prefix. Read as "가목", "나목"

### Citation Format (법제처 standard)
```
제10조제2항제3호가목  (no spaces — standard)
제10조 제2항 제3호 가목  (with spaces — also seen)
```

### Contract/ToS Variants
Contracts may also use:
- (1) (2) (3) instead of ① ② ③ for paragraphs
- (가) (나) (다) instead of 가. 나. 다. for sub-items

## Defined Term Convention

### At First Occurrence
```
비밀정보(Confidential Information, 이하 "비밀정보"라 한다)
```
Pattern: `target_term(source_term, 이하 "abbreviation"이라 한다)`

### Quotation Marks
- **법령 (statutes)**: 큰따옴표 " "
- **판결문 (court decisions)**: 작은따옴표 ' '
- **계약서/약관**: either " " or ' '

### Rules
- Abbreviation introduced at FIRST mention of full term
- 목적조항 (purpose article, Article 1) should NOT use abbreviations
- Subsequent occurrences: abbreviated term only, no re-definition

## Emphasis Convention

Korean has NO ALL CAPS equivalent. Map as follows:

| Source (EN) | Target (KO) |
|---|---|
| ALL CAPS text | **굵은 글씨 (Bold)** |
| Bold | Bold |
| Italics (case names) | Not applicable — use quotation marks or 「」 |
| Underline | 밑줄 (Underline) — traditional legal emphasis |

**약관규제법 제3조**: Important clauses in ToS MUST use "부호, 색채, 굵고 큰 문자 등" for emphasis. This is legally enforceable.

## Legal Citation Format

### Law Names: 「 」 (홑낫표, single angle brackets)
```
「개인정보 보호법」 제15조제1항
「민법」 제750조
```
**NOT** 『 』 (겹낫표). **NOT** < > or << >>.

### Law Number Format
```
법률 제16930호
```
Full: 「개인정보 보호법」(법률 제16930호, 2020. 2. 4. 일부개정)

### Court Case Citation
```
대법원 2018. 10. 25. 선고 2018다275017 판결
```
Format: [법원명] [YYYY. MM. DD.] 선고 [XXXX{부호}{번호}] [재판유형]

### Key Case Type Codes
| Code | Level | Type |
|---|---|---|
| 다 | 대법원 (3심) | 민사상고 |
| 도 | 대법원 (3심) | 형사상고 |
| 나 | 고등법원 (2심) | 민사항소 |
| 가합 | 지방법원 (1심) | 민사합의 |

## Date Format

```
XXXX년 XX월 XX일
```
Example: 2026년 3월 16일

In case citations: `2018. 10. 25.` (periods with spaces, trailing period)

## Key Legal Terminology Standards (법제처)

### 순화 (Simplified) Terms
| 순화 전 (Avoid) | 순화 후 (Prefer) | English |
|---|---|---|
| 하자(瑕疵) | 흠, 결함 | Defect |
| 기망(欺罔) | 속임 | Fraud |
| 해태(懈怠) | 게을리 함 | Negligence |
| 간주하다 | ~으로 본다 | Deemed as |
| 당해(當該) | 그 / 해당 | The relevant |

### Party Designation
Korean contracts use: **갑(甲)** = Party A, **을(乙)** = Party B

## Common Pitfalls: 번역투 (Translationese) to AVOID

| 번역투 (Avoid) | 자연스러운 한국어 (Prefer) | Origin |
|---|---|---|
| ~에 의하여 / ~에 의한 | ~으로 / ~에 따라 / ~에 따른 | Japanese ~によって |
| ~에 있어서 | ~에서 / ~에 / ~할 때 | Japanese ~において |
| ~로부터 | ~에서 / ~에게서 | English "from" |
| ~것으로 한다 | ~한다 | Japanese ~ものとする |
| ~하는 것이 가능하다 | ~할 수 있다 | English "possible to" |
| ~를 행하다 | ~하다 | Japanese ~を行う |
| ~에 대하여 (overuse) | ~을/를 / ~에 관하여 | Japanese ~に対して |
| ~함에 있어 | ~할 때 / ~하는 데 | Japanese ~するにあたり |
| ~하는 바에 의한다 | ~에 따른다 | Japanese style |

### Examples
- **Bad**: "국민투표**에 의하여** 결정한다" → **Good**: "국민투표**로** 결정한다"
- **Bad**: "공부함**에 있어서** 불편함" → **Good**: "공부하는 **데에** 불편함"
- **Bad**: passive overuse ("승인이 요구된다") → **Good**: active voice ("승인을 받아야 한다")
